# Salesforce Flow API (salesforce-flow)
The Salesforce Flow API enables developers to interact with and manage Salesforce Flow automation processes programmatically. This includes creating, updating, querying, and executing flows within Salesforce.

**URL:** [Visit APIs.json URL](https://developer.salesforce.com/docs/atlas.en-us.api_rest.meta/api_rest/resources_flow.htm)
